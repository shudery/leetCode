### Test
安装依赖
```
npm install
```
测试所有题目
```
npm test
```
测试单个题目是否通过，传入参数，如测试001题
```
n=001 npm test
```
测试单个题目是否通过，并且测试不同方法执行时间
```
n=001 npm start
```
新建一个题目目录模板
```
npm run new
```
