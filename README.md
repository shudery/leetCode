# leetCode
没事刷刷算法题： [leetcode](https://leetcode.com/problemset/algorithms/)

### Test
安装依赖
```
npm install
```
测试所有题目
```
npm test
```
测试单个题目是否通过，传入参数，如测试001题
```
n=001 npm test
```
测试单个题目是否通过，并且测试不同方法执行时间
```
n=001 npm start
```
